# Sorting Algorithm Visualizer 
 HTML, CSS, JavaScript, Sorting Algorithm
Developed an interactive web application to visualize sorting algorithms including Bubble Sort, Selection Sort, Insertion
Sort, Merge Sort, and Quick Sort.
Implemented dynamic array visualization with animation, allowing users to observe step-by-step sorting operations.
Added features such as custom array input, random array generation, start/pause/resume controls.
